# 星愿APP - 高学历轻社交应用

## 项目简介

星愿APP是一款专为18-29岁年轻人设计的高学历轻社交应用，主打轻松愉悦的社群氛围，让每个人都能在万千世界中找到属于自己的星辰大海。

## 核心功能

### 🔐 用户认证系统
- **多种注册方式**：支持微信、QQ、手机号注册
- **实名认证**：每个账号必须关联手机号
- **用户ID系统**：8位数字UID，用户可自行更改
- **身份验证**：头像认证、学历认证、身份认证

### 👤 个人资料管理
- **社交名片**：个人简介、生日星座、兴趣爱好、家庭情况、日常生活、学历工作、理想伴侣
- **相册功能**：支持照片上传和分享
- **个性化设置**：自定义昵称、头像、个性签名

### 🏠 首页推荐系统
- **精准推荐**：基于学历、身高、三观匹配的智能推荐，每日21人
- **星盘推荐**：根据生辰星座和当天日期解读星盘，每日最多5次
- **学习算法**：根据用户历史行为优化推荐

### 👥 联系人管理
- **好友列表**：匹配成功的好友管理
- **星盘陌生人**：神秘身份聊天，3天无互动自动消失
- **星缘系统**：每日聊天续星缘，累计星缘代表社群流行度

### 📱 动态分享
- **私域星圈**：仅好友可见的私密分享
- **公域星海**：公开可见的社区动态
- **匿名功能**：支持匿名问答和求助
- **隐私控制**：灵活的分组和可见范围设置

### ⭐ 星运系统
- **星缘计算**：基于聊天频率和互动质量
- **星运等级**：微弱、闪烁、明亮、璀璨
- **社交影响**：星运影响在社群中的受欢迎程度

## 技术架构

### 开发环境
- **语言**：Kotlin
- **UI框架**：Jetpack Compose
- **架构模式**：MVVM + Repository
- **依赖注入**：Dagger Hilt
- **数据库**：Room
- **网络请求**：Retrofit
- **最低SDK**：API 24 (Android 7.0)
- **目标SDK**：API 34 (Android 14)

### 项目结构
```
app/
├── src/main/java/com/starwish/app/
│   ├── data/                    # 数据层
│   │   ├── model/              # 数据模型
│   │   ├── dao/                # 数据访问对象
│   │   ├── database/           # 数据库配置
│   │   └── repository/         # 数据仓库
│   ├── ui/                     # UI层
│   │   ├── screens/            # 界面
│   │   ├── navigation/         # 导航
│   │   └── theme/              # 主题样式
│   ├── viewmodel/              # 视图模型
│   └── di/                     # 依赖注入
└── src/main/res/               # 资源文件
```

## 安装和运行

### 环境要求
- Android Studio Arctic Fox 或更高版本
- JDK 8 或更高版本
- Android SDK API 24+

### 构建步骤
1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 等待Gradle同步完成
4. 连接Android设备或启动模拟器
5. 点击运行按钮

### 命令行构建
```bash
# 构建Debug版本
./gradlew assembleDebug

# 构建Release版本
./gradlew assembleRelease

# 运行测试
./gradlew test
```
